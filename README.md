# 宿舍電費追蹤

東華大學宿舍電費記錄與用量分析工具。不定時記錄剩餘電費餘額，自動推算本月用電度數、費率級距與平均花費。

**線上使用**：[https://42444977.github.io/NDHU_Electricity/](https://42444977.github.io/NDHU_Electricity/)

---

## 功能

- **記錄餘額**：輸入目前剩餘電費金額，自動附上時間戳
- **推算度數**：根據東華計費規則，從花費金額反推本月已用幾度電
- **費率級距**：顯示目前在哪個計費區間，以及距下一級距還有多少度
- **平均用量**：計算每小時、每天平均費率，並預測本月總花費
- **充值偵測**：偵測到餘額增加時顯示提示，建議充值前後各記一筆以確保精確
- **趨勢圖表**：折線圖以等距時間刻度顯示餘額變化，根據資料範圍自動選擇小時或天為單位
- **歷史記錄**：所有讀數列表，可逐筆刪除

---

## 計費規則（每月 1 日重置）

| 用電量 | 費率 |
|--------|------|
| 0 – 110 度 | $2.1 / 度 |
| 111 – 700 度 | $2.68 / 度 |
| 701 度以上 | $3.97 / 度 |

---

## 使用方式

1. 打開線上網址（或本地的 `index.html`）
2. 查看宿舍電卡剩餘餘額
3. 在「新增讀數」欄位輸入金額，按 **記錄** 或 Enter
4. 重複以上步驟，累積兩筆以上即可看到統計數據

> **充值時的建議**：儲值前先記錄一次，儲值完成後立刻再記錄一次，避免用電量被漏算。

---

## 資料儲存

所有記錄存在瀏覽器的 `localStorage`，不上傳至任何伺服器。換裝置或清除瀏覽器資料後記錄會消失，請注意備份。

---

## 本地執行

無需任何安裝，直接用瀏覽器開啟 `index.html` 即可。
